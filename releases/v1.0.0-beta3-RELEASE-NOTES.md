## üéâ What's New

### Double-Click Installer
- **No more manual installation!** Just double-click `Install Reflect.command` inside the DMG
- Automatically installs both the driver and app
- Handles admin permissions, Gatekeeper bypass, and audio system restart
- Offers to launch Reflect when installation completes

### Improved Menu Bar Icon Visibility
- **Fixed:** Menu bar icon is now **white** when inactive (was barely visible gray)
- **Active processing:** Icon turns **orange/amber** for clear visual feedback
- **Bypassed:** Icon shows gray with slash
- **Error state:** Red icon with exclamation mark

### Uninstaller Included
- Clean removal with `Uninstall Reflect.command`
- Removes app, driver, and optionally preferences
- Restores audio system to default state

## ‚ö†Ô∏è Important Notes

### Unsigned Software
This is an **unsigned beta release**. On first run:
1. **Right-click** `Install Reflect.command` ‚Üí **Open**
2. Click **Open** in the security dialog
3. macOS will remember this choice

### Corporate/Managed Macs
If your Mac is managed by IT (MDM), unsigned apps may be blocked entirely. This beta is intended for **personal Macs** or unmanaged systems.

### System Requirements
- macOS 13 (Ventura) or later
- Apple Silicon or Intel Mac
- Administrator access
